# 文本（text）

可以指定文本显示的行数。

## 使用

```js
import Vue from 'vue'
import { Text } from 'tms-vue-ui'
Vue.use(Text)
```

## 属性（props）

| 参数     | 说明                               | 类型   | 默认值 |
| -------- | ---------------------------------- | ------ | ------ |
| lines    | 文本显示的行数                     | number | -      |
| lines-sm | 小屏幕下（小于等于 768）显示的行数 | number | -      |
